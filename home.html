About the Game

Modern City Environment

Set in a lively city (think Manhattan!), brace yourselves as Free Run takes you on a quick-paced ride across town.

Salient Features

Dynamic Game World

We've randomized the rendering of the game world, so that you never get bored while playing.

Animations

Lively animations are configured to enhance your interaction with the game environment.

Increasing Difficulty

We gradually increase the difficulty of the game as the player moves further in the run so as to make it more challenging.
